Blink-Controlled Smart Glasses using EOG

Project Overview

Developed an interactive wearable project using the BIOAmp EXG Pill by Upside Down Labs that enables hands-free photography through eye blinks. The system leverages Electrooculography (EOG) to detect intentional eye blink patterns and trigger image capture.

The smart glasses continuously monitor eye movement signals, and when the capture mode is activated, a deliberate blink is recognized as a command to take a photograph of the user’s field of view. This project demonstrates the practical application of biopotential signal acquisition in human–computer interaction

Key Features

  • EOG-based eye blink detection using BIOAmp EXG Pill
  • Blink-triggered image capture mechanism
  • Wearable smart-glasses form factor
  • Hands-free and intuitive user interaction.

Interactive Demonstration

To validate and showcase the system’s real-time performance, a custom Flappy Bird game was developed in Unity, controlled entirely using muscle signals from MyoFlex. This interactive application demonstrated reliable EMG-based control and highlighted the device’s potential in human–computer interaction (HCI) and assistive technologies.
