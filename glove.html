Sign-to-Speech Smart Glove

Project Overview

Developed a Sign-to-Speech Smart Glove that translates hand gestures and sign language into audible speech, enabling more accessible communication for individuals with speech and hearing impairments. The system uses a combination of flex sensors and IMU data to accurately capture finger movements and hand orientation. Gesture data is transmitted to a Firebase backend, where it is stored and made available for real-time processing.

A Flutter-based mobile application retrieves the gesture data from Firebase, processes it, and converts recognized signs into spoken output, providing an intuitive and portable communication interface.

Key Features

  • Real-time sign language and gesture recognition.
  • Flex sensors + IMU for precise hand movement detection
  • Flutter mobile app for gesture processing and speech generation
  • Firebase integration for data transfer and synchronization.

Impact & Applications

This project demonstrates the effective integration of wearable sensors, cloud-based data handling, and mobile applications to address real-world accessibility challenges. Potential applications include: Assistive communication devices, Augmentative and Alternative Communication (AAC) systems, Human–computer interaction research and Inclusive technology solutions
