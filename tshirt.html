Smart Posture-Correcting T-Shirt with Haptic Feedback

Project Overview

Developed a smart posture-correcting wearable t-shirt that combines IMU-based posture tracking with real-time haptic feedback to help users maintain proper posture throughout daily activities. The system continuously monitors body orientation using an Inertial Measurement Unit (IMU) and detects posture deviations in real time. When incorrect posture is identified, integrated haptic motors gently alert the wearer, enabling immediate correction without visual distraction.

A custom calibration feature allows users to define personalized posture baselines and correction thresholds, ensuring adaptability across different body types and use cases. The t-shirt connects seamlessly to a smartphone application, providing detailed posture analytics, progress tracking, and real-time notifications to support long-term posture improvement.

Key Features

  • IMU-based posture monitoring with high accuracy.
  • Real-time haptic feedback for posture correction.
  • User-specific calibration for personalized posture goals.
  • Mobile app integration for insights, tracking, and alerts.
